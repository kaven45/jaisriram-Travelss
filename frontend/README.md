# Frontend

Angular frontend for JaiSriRam Travels application.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:4200`

## Features

- Customer Management
- Driver Management
- Vehicle Management
- Booking Management

## API Integration

The frontend communicates with the backend API at `http://localhost:8080/api`

## Build

To build for production:
```bash
npm run build
```

The build artifacts will be stored in the `dist/frontend` directory.
